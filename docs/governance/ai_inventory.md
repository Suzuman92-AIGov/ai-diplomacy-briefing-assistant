# AI Inventory Entry

## AI System Name

RAG-based AI Foreign Policy Briefing Assistant

## Purpose

To support monitoring, summarization and analysis of public-source developments in AI governance, digital policy and international relations.

## AI Type

Retrieval-Augmented Generation system using public sources and LLM-based summarization.

## Decision Impact

Informational support only. The system does not make policy decisions and does not publish content automatically.

## Required Controls

- Public sources only
- Source citations
- Source reliability tiers
- Human review before external use
- Senior review for sensitive topics
- Audit logging
- No-source-no-claim rule
