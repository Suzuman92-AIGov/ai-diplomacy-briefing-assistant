# Governance Workflow

```text
Define purpose
  -> Approve source list
  -> Ingest public sources
  -> Generate chunks and embeddings
  -> Retrieve source-grounded context
  -> Generate draft brief
  -> Classify sensitivity
  -> Human review
  -> Senior review for sensitive items
  -> Internal distribution
  -> Optional adaptation for public communication
  -> Audit log and periodic review
```

## Review statuses

- Draft
- Reviewed
- Approved
- Rejected
- Needs senior review
